Glossary (quick revision)

Short, plain-language definitions for all key terms in this report.

Blockchain
A ledger of linked blocks that is hard to alter retroactively.
Bitcoin
A decentralized network focused on peer-to-peer digital money; native coin is BTC.
Stablecoin
A token designed to stay near a reference value (often USD).
Ethereum
A programmable blockchain supporting smart contracts and tokens.
Smart Contract
Program code deployed on-chain that executes deterministic rules.
EVM
Ethereum Virtual Machine: runtime that executes Ethereum smart contracts.
Decentralization
Control spread across many participants instead of one central authority.
Transparency
Transaction/state data can be publicly verified.
Immutability
Past records become difficult and costly to modify.
Node
Software instance that stores/verifies blockchain data and relays messages.
On-chain
Action/data recorded directly on blockchain.
Off-chain
Action/data handled outside blockchain systems.
Wallet
Tool for managing keys, addresses, and transaction signing.
Wallet Address
Public identifier used to receive assets.
Public Key
Shareable cryptographic key used for verification.
Private Key
Secret cryptographic key used to sign and authorize actions.
Secret Recovery Phrase
Backup phrase that can regenerate wallet keys.
Signature
Cryptographic proof that message/transaction was approved by key owner.
Transaction
Signed instruction to transfer value or call contract logic.
Transaction Fee
Cost paid to include/execute transaction on network.
Gas
Unit measuring computational work on Ethereum-like systems.
Mempool
Waiting area of pending transactions seen by nodes.
Mine Block
PoW process of creating a valid block by computational work.
Sign Block
PoS validator attestation/proposal confirming block validity.
Consensus
Protocol process that decides canonical chain history.
Token
Asset created by smart contract on an existing blockchain.
Coin
Native asset of a blockchain protocol.
Explorer
Interface for viewing blocks, transactions, and addresses.
P2P Exchange
Direct person-to-person crypto trading marketplace.
OTC Exchange
Desk-style negotiated trading, often for larger orders.
